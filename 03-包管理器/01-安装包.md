# 安装包

## 测试步骤

1. Window -> Show View -> Other -> RuyiSDK -> Ruyi Package Explorer
2. 选择 milkv-duo
3. 在底栏找到 Ruyi Package Explorer
4. 勾选 toolchain -> gnu-plct -> latest
5. 点击 Download 按钮

## 预期结果

成功完成安装，输出 `RUYI_DONE`

如需验证，可以执行 `~/.local/bin/ruyi list --is-installed 'y' --name-contains 'gnu-plct'`
