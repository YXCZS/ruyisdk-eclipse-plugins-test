# 安装包

## 操作步骤

1. 点击Window -> Show View -> Other -> RuyiSDK -> Ruyi Package Explorer进入包管理器
2. 选择开发板
3. 选择需要安装的包后后点击Install
4. 命令行和视图同步查看安装情况

## 预期结果

能够正常安装包
